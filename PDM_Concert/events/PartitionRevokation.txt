#Basically, for this I'd planned to have a decision that specifically the Polish-Lithuanian Commonwealth can take when it's a Great Power. It fires a "Revocation of the Partitions" major event, which in turn sends an event to each of the Great Powers with two options: to support Polish independence, or to maintain the partitions. Germany, Austria, and Russia will almost never support independence, except Austria will, if it's a democracy. The USA is generally inclined to support independence, while Britain can go either way, depending on its relations with Germany or Russia. Italy will also support independence if Austria still owns any of its cores, but otherwise is similar to Britain. France, at least, will almost always support independence, unless it's in the weird case of being allied to Germany somehow. Any country allied to the PLC will support independence, as well.
#
#However, since the PLC itself doesn't get a vote in the conference even if it's a Great Power, the deck's generally stacked against the revocation of the partitions - after all, you've got three almost guaranteed "No" votes, with only one more needed to prevent independence completely. This makes the political situation of the revocations interesting, because as the PLC, you have to really examine your geopolitical situation: you could choose to make Austria weak and collapse, removing it from the opposition, but opening up the Balkans for further Russian influence. You could weaken the Russians themselves, but a weak Russia would inspire Byelorussian and Ukrainian/Ruthenian ideas of self-governance, serious threats to the territorial integrity of the Commonwealth. The Ottomans, too, might not support the revocation of the partitions; they have a less-than-friendly history with the Polish, and even if you propped them up and helped them continue to be a Great Power, they might see this as an opportunity to keep you weak: better the enemy you know - Russia - than the friend you don't - the newly-revived Polish state.
#
#I meant to include all this way back when, but I ended up doing everything except for the Revocation of the Partitions, since it's just so complex. I'll probably revisit it later with all the attention it deserves; I'm planning on adding a general "Conference" event chain for countries seeking full independence, in the case of all their cores being cores of another, larger nation. This is something I feel should have been addressed with the crisis system of HoD, but unfortunately wasn't; if, say, the Great Powers deem that Andalusia should be an independent state, then as it is now, Spain will just march in a decade later and reconquer those territories - it's kind of silly.

country_event = {
	id = 1
	title = "EVTNAME1" #The Partition Question
	desc = "EVTDESC1"
	picture = "greatpowers"

	trigger = {
		is_greater_power = yes
		#capital_scope = { continent = europe }
		NOT = {
			#Not PLC itself
			tag = PLC
			#None of the partitioners
			tag = PRU
			tag = NGF
			tag = GER
			tag = RUS
			tag = AUS
			tag = KUK
			tag = DNB
			has_global_flag = partition_revokation_conference_held
			has_global_flag = partition_revokation_conference_in_progress
			has_country_flag = will_not_hold_partition_revokation_conference
			has_country_modifier = neutrality
		}
		PLC = { exists = yes }
		has_global_flag = raised_partition_revokation
	}

	mean_time_to_happen = {
		days = 28
		modifier = {
			factor = 0.5
			ai = no
		}
	}

	option = {
		name = "We will host the conference."
		prestige = 5
		set_global_flag = partition_revokation_conference_in_progress
		set_country_flag = hosting_partition_revokation_conference
		PLC = {
			set_variable = {
				which = partition_revokation_conference_revoke
				value = 0
			}
			set_variable = {
				which = partition_revokation_conference_uphold
				value = 0
			}
		}
		any_country = {
			limit = {
				OR = {
					AND = {
						is_greater_power = yes
						capital_scope = { continent = europe }
					}
				}
			}
			country_event = 2
		}
		THIS = { country_event = 2 }
		ai_chance = {
			factor = 50
		}
	}

	option = {
		name = "This is unnecessary."
		set_country_flag = will_not_hold_partition_revokation_conference
		prestige = -2
		ai_chance = {
			factor = 50
			modifier = {
				factor = 0
				tag = ENG
			}			
		}
	}
}
